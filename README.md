# Ingaia backend-challenge

Api para calcular e retornar o valor de um imóvel com base no tamanho informado e no custo do metro quadrado.

## Pré-requisitos
`node v14.15.2` [download](https://nodejs.org/) 

`npm 6.14.9`

## Swagger
Disponível [aqui.](https://app.swaggerhub.com/apis-docs/M7149/ingaiachallenge/1.0.0) 

## Heroku
Disponível [aqui.](https://ingaia-challenge-backend.herokuapp.com/) 

## Instalação

### Clone o repositório abaixo.

` 
git clone https://github.com/matheusmacari/ingaia-backendchallenge.git
`

### Instale as dependências com o seguinte comando
`npm install`

### no diretório do repositório, execute o comando para iniciar o servidor da API
`node server.js`


## Testando a API
### Abra seu navegador local e tente acessar     
`https://localhost:3333/api1/`
